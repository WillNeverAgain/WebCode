- **入口与导航架构**：以 `index.html` 为单页应用（SPA）核心，采用垂直全屏滚动（`pages-wrapper` + `translateY`）展示11个世界观板块；通过顶部导航栏切换至 `pages/characters.html` 和 `pages/stories.html` 的 iframe 沉浸式视图。
- **内容组织模式**：`pages/` 目录下包含独立的故事章节（如 `乡下人的悲歌.html`），采用“场景（Scene）”驱动的线性叙事结构，通过 JavaScript 控制 DOM 显隐实现类似视觉小说的阅读体验。
- **资源管理策略**：多媒体资源（图片、音频）直接散落在根目录及 `pages/` 文件夹中，通过相对路径在 HTML 中硬编码引用；背景音效通过 `localStorage` 持久化播放状态，实现跨页面续播。
- **样式隔离与复用**：主站采用自定义 CSS 实现复古胶片风格；子页面（如角色页）引入 Tailwind CSS CDN 进行快速布局，并通过内联 `<style>` 定义特定的动画与响应式规则。
